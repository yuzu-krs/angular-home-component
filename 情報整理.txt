##

- angular.json
- Angular CLI の設定ファイル
- ビルドやサーブの設定、アセットやスタイルの読み込み、プロジェクト構成を定義
- 例: どの CSS をグローバルに読み込むか、ビルド出力先など

- .editorconfig
- エディタ共通のコードスタイル設定
- インデント幅、改行コード、文字コードなどを統一するためのファイル

- index.html
- アプリの最初に読み込まれる HTML
- <app-root></app-root> が置かれていて、Angular コンポーネントがここに展開される
- メタタグや外部ライブラリの読み込みもここで設定

- main.ts
- Angular アプリの起動処理を記述
- bootstrapApplication(AppComponent) などでルートコンポーネントを起動する
- 開発時と本番時の環境設定を切り替えることもある

- styles.css
- プロジェクト全体に適用される CSS
- 各コンポーネントの CSS とは別に、共通デザインやリセット CSS を記述

- app.ts
- ルートコンポーネント (AppComponent) の TypeScript ファイル
- アプリ全体のロジックや状態を持つ最上位コンポーネント

- app.routes.ts
- ルーティング設定ファイル
- URL パスとコンポーネントの対応関係を定義

- app.html
- AppComponent のテンプレート (HTML)
- ナビゲーションバーや <router-outlet> を配置して、画面遷移の基盤を作る

- app.css
- AppComponent 専用のスタイル
- グローバルではなく、このコンポーネントにだけ適用される CSS

- app.config.ts
- アプリの設定情報をまとめるファイル
- API のエンドポイント、環境変数、サービス設定などを管理
- プロジェクトによっては environment.ts を使う場合もある

##

 (ngSubmit)="submitApplication()"
  フォームの送信イベントを Angular がキャッチして submitApplication() メソッドを呼び出します。
  これは通常の HTML の submit ではなく、Angular 独自の ngSubmit です。
  FormsModule をインポートしていないと ngSubmit は使えません。

  [(ngModel)]="name"
  双方向データバインディングです。
  ユーザーが入力した値をコンポーネント側の name 変数に自動で反映します。
  逆にコンポーネント側で name を書き換えると、入力欄にも反映されます。
  これも FormsModule が必要です。

- フォーム入力
- ユーザーが <input> に文字を入力すると、[(ngModel)]="name" や [(ngModel)]="email" によって、コンポーネントの name / email プロパティに即座に反映されます。
- これは 双方向データバインディングなので、逆にコンポーネント側で this.name = 'Imai'; と書けば、入力欄にも "Imai" が表示されます。
- フォーム送信
- <form (ngSubmit)="submitApplication()"> によって、送信ボタンを押すと Angular が submitApplication() メソッドを呼び出します。
- このとき、すでに name と email プロパティには入力値が入っているので、メソッド内でその値を評価できます。
- メソッド内で評価

アプリケーション内で、あるコンポーネントから別のコンポーネントへ画面遷移させる仕組み。
SPAではページ全体を再読み込みせず一部だけ更新。
Angular Router
URLパスを表示するコンポーネントを対応付ける仕組み。

##

#filterのテンプレート変数とは
1.#filter という名前でこのinput要素を参照できるようなる。
<input type="text" placeholder="Filter by city" #filter />

2.ボタンがクリックされたとき、filter.value(inputの値)をfilterResults()メソッドに渡す。
<button class="primary" type="button" (click)="filterResults(filter.value)">Search</button>

3.コンポーネント側のfilterResult(text: string)が、その値を使って絞り込み処理を実行。
